## 🚀 Lancement local

Aucune dépendance à installer. Il suffit d'ouvrir le fichier directement dans un navigateur :

```bash
# Option 1 — ouvrir directement
open index.html

# Option 2 — lancer un serveur local (recommandé pour éviter les problèmes CORS)
npx serve .
# ou
python -m http.server 8080
```
